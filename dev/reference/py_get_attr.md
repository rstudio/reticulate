# Get an attribute of a Python object

Get an attribute of a Python object

## Usage

``` r
py_get_attr(x, name, silent = FALSE)
```

## Arguments

- x:

  Python object

- name:

  Attribute name

- silent:

  `TRUE` to return `NULL` if the attribute doesn't exist (default is
  `FALSE` which will raise an error)

## Value

Attribute of Python object
